Flip Meet

Choose which sites /meet should activate on. The extension watches the chat composer at /chats/* on each allowed site and replaces /meet with a fresh Google Meet link.

Default sites

Built into the extension. Toggle these off via chrome://extensions if you need to.

    Custom sites

    Add your own. Useful for new Flip environments (e.g. prod.flipnext.de) or any other chat tool that uses a /chats/* URL pattern.

      After adding, refresh any open chat page on that site to activate the extension there.
